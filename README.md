# DroneLog – Startseite (Mock)

Ordnerstruktur:
- index.html
- assets/
  - css/style.css
  - js/applyTheme.js (lädt assets/data/theme.json und setzt CSS Variablen)
  - js/main.js (Mobile-Menü Toggle)
  - data/theme.json (DEIN "Style" im JSON)
  - img/ (Logo + 5 Platzhalterbilder)

Start:
- Datei `index.html` im Browser öffnen (oder per Live-Server).

Anpassungen:
- Farben/Schriften/Radien in `assets/data/theme.json` ändern.
- Platzhalterbilder in `assets/img/feature-*.png` durch echte Screenshots ersetzen.
- Footer-Links (Impressum/Datenschutz/AGB) auf eure echten Seiten routen.
